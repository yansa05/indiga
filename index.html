TENTANG KAMI

Saya yansa

Saya sangat menyukai game. Saat masih kuliah saya belajar membuat game dan mencoba membangun sebuah game horror PC dengan judul Hantu Survival. Tahun 2019 saya mencoba merilis game mobile pertama saya bersamaan dengan nama tim dev Indiga, sayangnya game tersebut gagal atau jumlah downloadnya sangat sedikit. Tahun 2020 saya mencoba merilis game saya waktu kuliah yaitu Hantu Survival. Hingga saat ini Hantu survival sudah di mainkan oleh 50 ribu lebih pengguna android. Dan sekarang Indiga mencoba untuk membuat versi 2 nya. Ditunggu ya. Lets have Fun
